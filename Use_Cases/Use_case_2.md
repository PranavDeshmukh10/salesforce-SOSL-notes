## 🧩  Use Case — Find all records with the name Pranav Deshmukh

**Description:**
Search across all objects for records where the name matches `Pranav Deshmukh`. The `IN Name FIELDS` clause restricts the search to name fields only.

## 🛠️ Solution
```sql
FIND {Pranav Deshmukh} IN Name FIELDS
```
